# mlx-gpt-example
A GPT implementation purely using Apple's MLX framework, adapted from Andrej Karpathy's "Let's build GPT" tutorial.

## Dependencies
- Python >= 3.11
- Apple Silicon Mac (for GPU acceleration)
- Dependencies: `mlx>=0.29.3`, `numpy>=2.3.4`

## Quickstart
1. Install depedendencies
```bash
pip install -e
```

2.  Train the model
```bash
python gpt_example.py
```
